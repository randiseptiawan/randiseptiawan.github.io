Back End Web

Banking Innovation CoCreate Platform

  • Finished
  • September 2021
  • October 2021

Project ini dilakukan untuk menyelesaikan Final Project pada pelatihan Thematic Academy Backend IT Perbankan Digital Talent Scholarship oleh KOMINFO menggunakan Bahasa Pemrograman Golang. Project ini bertujuan untuk membuat sebuah platform untuk karyawan pada suatu bank yang ingin menyalurkan dan mengembangkan ide dan inovasinya melalui project ataupun artikel. Fitur dalam project ini terdiri dari Login dan Register menggunakan JWT Authentication, data disimpan pada database menggunakan MySQL, kemudian terdapat halaman beranda dimana user dapat mengakses project maupun artikel yang sudah dibuat user lain maupun membuat project atau artikel baru.
